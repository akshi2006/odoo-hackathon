Users can:
- create trips
- add cities and activities
- manage budgets
- make packing checklists
- share trip plans

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Node.js
- Express

Database:
- PostgreSQL

Features
User authentication
Trip planning
Multi-city itineraries
Budget tracking
Packing checklist
Shareable trip links
Responsive UI


## FOLDER STRUCTURE 
client/    -> frontend
server/    -> backend API
database/  -> database related files
docs/      -> notes and planning
